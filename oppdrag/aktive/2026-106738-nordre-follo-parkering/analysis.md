# Nordre Follo parkeringsfagsystem — vurdering

## TL;DR

**Klart NO-GO.** SSA-L + SaaS + mettet produktmarked = hylleprodukt-
anskaffelse, ikke utvikling. Nordre Follo vil kjøpe lisens på eksisterende
parkerings-SaaS (Aimo Park, EasyPark, Parkman, Europark, APCOA), ikke la
noen bygge fra scratch. Kontakt Tove Seigerud Sjølund.

## Avgjørende sitat fra UBL

> "Oppdragsgiver ønsker å benytte seg av skybaserte løsninger (SaaS), og
> vil bruke statens standard avtale SSA-L ved signering av kontrakt."

SSA-L = lisens/leie på leverandørens eksisterende hylleprodukt. Det er
ikke rammeverk for custom dev.

## Fakta

- **Kunde:** Nordre Follo kommune, Akershus (~60k innbyggere)
- **Scope:** Mobilapp for trafikkbetjenter, ileggelses­administrasjon,
  integrasjon mot saksbehandling + regnskap, HC-kort for forflytnings­
  hemmede
- **Prosedyre:** Prekvalifisering → konkurranse med forhandling
- **Tildelingskriterier:** 70 % kvalitet, 30 % pris
- **Kontrakt:** 4 år (36 mnd planperiode)

## Markedsbilde

Etablert, modnet marked:

| Leverandør | Posisjon |
|---|---|
| Aimo Park (tidl. Q-Park) | Nordisk storspiller, Oslo + flere byer |
| EasyPark | Europa, 21+ land |
| Parkman | Skandinavisk administrasjons-suite |
| APCOA / Europark | Enterprise |
| Metric, PDX, Nivel | Mindre integrasjons­spillere |

Alle har norske referanser, ferdig integrasjoner mot SAP/Visma/Oracle,
24/7 support, GDPR-dokumentasjon.

## Fit-analyse — Asbjørn

- Ingen parkerings-domene-referanse
- Solo kan ikke tilby langdrift-SLA som kommune forventer for 4-års SaaS
- Referanse-krav (andre kommuner med tilsvarende løsning i drift) er
  umulig å oppfylle
- Evaluering vekter produktmodenhet og kundereferanser høyt (en del av
  70 % kvalitet)

## Konklusjon: NO-GO

Ikke send prekvalifiseringssøknad. SSA-L + SaaS i mettet marked er
aldri solo-leverbar. Bruk tiden på antidoping-appen og eventuelt
eksportfinansiering-nettsider.
