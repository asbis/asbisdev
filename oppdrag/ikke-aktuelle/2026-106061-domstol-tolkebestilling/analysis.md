# Domstol tolkebestillingsløsning — vurdering

## TL;DR

**NO-GO som solo primærleverandør.** DA søker drift + vedlikehold av skybasert
tolkebestillings­løsning — trolig erstatning/videreføring av Proweb-systemet.
5M NOK statlig kontrakt + SSA-L + ubegrenset avtaleperiode signaliserer at
kjøper forventer etablert leverandør med driftsteam, ISO-sertifisering og
leveringssikkerhet. Solo-konsulent er ikke troverdig her. Eneste realistiske
vinkel er som underleverandør til Bouvet/Sopra Steria eller partner med
etablert tolkeformidler (Salita, Noricom, Hero Tolk).

## Fakta

- **Kunde:** Domstoladministrasjonen (60+ domstoler)
- **Estimert verdi:** 5M NOK
- **Frist:** 30. april 2026 10:00
- **Prosedyre:** neg-w-call, minst 3 kortlistes
- **Avtaleperiode:** Ubegrenset
- **Eksisterende system:** Proweb (tolkeregistrering + bestilling) +
  Intrprtr for politiattest-validering

## Kvalifikasjonskrav (fra UBL)

1. Årsregnskap 2024–2025 med revisjonsberetning (finansiell styrke)
2. Lovlig etablert foretak
3. Referanse: "Etablering, drift og vedlikehold av skybasert løsning for
   bestilling av fagpersonell" — **Asbjørn har ikke denne referansen**
4. Dokumentert kvalitetsledelsessystem (ISO 9001-nivå)
5. Gjennomføringsevne + organisering (inkl. underleverandørforhold)

## Sperrer for solo

- **Finansiell/organisatorisk:** 5M NOK statlig → kjøper vil vekte
  soliditet og driftsorganisasjon høyt
- **Domene-referanse:** Ingen referanse innen staffing/matching-plattformer
- **ISO/sikkerhetsledelse:** Ingen formell sertifisering som firma
- **SLA-risiko:** Enmanns-bemanning kan ikke garantere 24/5-respons på
  rettskritisk system
- **Konkurrenter:** Bouvet, Sopra Steria, Knowit + etablerte
  tolkeformidlere med egne plattformer (Salita, Semantix, TolkeNett)

## Fit-analyse

**Styrker:** Go-backend, PCI DSS (EaseePay), integrasjons­erfaring,
mobile-frontend for eventuell tolk-app.

**Svakheter:** Ingen drifts­erfaring på statlig 5M+ kontrakt, ingen HR/
matching-domene, ingen ISO-sertifisering.

## Anbefaling

**NO-GO som primær.** Vurder betinget inngang som *underleverandør* til
et etablert IT-hus (kontakt Bouvet Oslo/Sopra Steria denne uken) hvis
bud-teamet der søker teknisk partner. Ellers arkiver og prioriter
antidoping-appen og andre solo-leverbare.
